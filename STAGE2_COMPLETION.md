## 🔄 Рабочий процесс

```
Документ DOCX
    ↓
[ValidationEngine стартует]
    ↓
1️⃣  StructureValidator (обязательно первым)
    Проверяет: содержание, разделы, объём, нумерацию страниц
    ↓
2️⃣  FontValidator → MarginValidator → ParagraphValidator → HeadingValidator
    Проверяют: шрифты, поля, абзацы, заголовки (параллельно)
    ↓
3️⃣  BibliographyValidator ← TableValidator ← FormulaValidator
    Проверяют: библиография, таблицы, формулы (параллельно)
    ↓
[Агрегирование результатов]
    ↓
[Генерация отчёта с рекомендациями]
    ↓
JSON результат
```

## 🎓 Примеры использования

### Python API

```python
from app.services.validation_engine import ValidationEngine

profile = {
    'name': 'ГОСТ 7.32-2017',
    'rules': {
        'formulas': {
            'require_numbering': True,
            'require_references': True
        }
    }
}

engine = ValidationEngine(profile=profile)
report = engine.validate_document('document.docx')

print(f"Статус: {report['status']}")
print(f"Проблем: {report['summary']['total_issues']}")
```

### REST API

```bash
curl -X POST http://localhost:5000/api/validation/check \
  -F "file=@document.docx" \
  -F "profile_name=gost_7_32_2017"
```
