Спасибо за интерес к проекту CURSA! Коротко — как внести вклад:

1. Форкните репозиторий и создайте ветку `feature/имя` или `fix/имя`.
2. Следуйте существующему стилю кода. Для Python — PEP8; для JS — используйте проектный стиль (MUI + functional components).
3. Коммиты: короткое сообщение в заголовке, при необходимости — тело с подробностями. Пример: `feat: add profile import endpoint`.
4. Перед PR запустите тесты: `cd backend && .venv\Scripts\Activate.ps1 && python -m pytest`.
5. Описывайте PR: что сделано, почему, как протестировать.
6. Для крупных изменений создайте issue перед началом работы.

Спасибо! Все PR рассматриваются и проходят CI перед слиянием.
