## 🚀 Как использовать

### Мгновенный старт

```bash
cd backend
python quick_validation_test.py
```

Создаст тестовый документ и запустит полную валидацию.

### Через API

```bash
curl -X POST http://localhost:5000/api/validation/check \
  -F "file=@document.docx" \
  -F "profile_name=gost_7_32_2017"
```

### В коде Python

```python
from app.services.validation_engine import ValidationEngine
from docx import Document

# Загружаем документ
doc = Document('document.docx')

# Создаём движок валидации
engine = ValidationEngine(profile='gost_7_32_2017')

# Выполняем полную проверку
result = engine.validate_document('document.docx')

# Обрабатываем результат
if result['passed']:
    print("✅ Документ соответствует требованиям")
else:
    for issue in result['issues']:
        print(f"❌ {issue['description']}")
```
