## Слайд 6: Алгоритм работы

```
┌─────────────────────────────────────────────────────────────┐
│                 АЛГОРИТМ РАБОТЫ СИСТЕМЫ                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│     1. ПОЛЬЗОВАТЕЛЬ                                        │
│        Загружает DOCX файл + выбирает профиль               │
│                    │                                        │
│                    ▼                                        │
│     2. BACKEND                                              │
│        Сохраняет файл → Запускает Validation Engine         │
│                    │                                        │
│                    ▼                                        │
│     3. VALIDATION ENGINE                                    │
│        Последовательно запускает 10 валидаторов              │
│        Каждый проверяет свой аспект документа                │
│                    │                                        │
│                    ▼                                        │
│     4. РЕЗУЛЬТАТ                                           │
│        Формируется отчёт:                                   │
│        • Список ошибок с позициями                          │
│        • Оценка соответствия (0-100%)                       │
│        • Рекомендации по исправлению                        │
│        • Возможность автокоррекции                          │
│                    │                                        │
│                    ▼                                        │
│     5. ПОЛЬЗОВАТЕЛЬ                                        │
│        Получает отчёт → Скачивает исправленный файл          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
