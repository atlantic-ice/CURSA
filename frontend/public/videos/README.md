# Видео-инструкции для ручных исправлений

Добавьте MP4 файлы в эту директорию для показа в руководствах на странице отчёта.

## Необходимые видео файлы:

| Файл | Описание |
|------|----------|
| `add_section.mp4` | Добавление обязательных разделов (Введение, Заключение и т.д.) |
| `title_page.mp4` | Оформление титульного листа |
| `table_of_contents.mp4` | Создание автоматического содержания |
| `fix_font.mp4` | Исправление шрифта (Times New Roman, 14 пт) |
| `fix_margins.mp4` | Настройка полей страницы (3-1.5-2-2 см) |
| `fix_line_spacing.mp4` | Настройка межстрочного интервала (полуторный) |
| `fix_indent.mp4` | Настройка абзацного отступа (1.25 см) |
| `fix_bibliography.mp4` | Оформление списка литературы по ГОСТ |
| `fix_figure.mp4` | Оформление рисунков и подписей к ним |
| `fix_table.mp4` | Оформление таблиц и их заголовков |
| `fix_page_numbers.mp4` | Настройка нумерации страниц |
| `fix_headings.mp4` | Оформление заголовков (стили Heading 1/2/3) |

## Рекомендации по записи видео:

1. **Разрешение**: 1280x720 (720p) или 1920x1080 (1080p)
2. **Формат**: MP4 с кодеком H.264
3. **Длительность**: 30-90 секунд
4. **Содержание**: 
   - Показать открытие Word
   - Демонстрация каждого шага из инструкции
   - Финальный результат

## Как добавить видео в руководство:

В файле `ReportPage.js` найдите компонент `ManualFixGuide` и замените placeholder:

```jsx
{/* Вместо placeholder */}
<video 
  src={guide.videoPlaceholder} 
  controls 
  style={{ width: '100%', borderRadius: 8 }}
  poster="/videos/poster.png"
/>
```

## Альтернатива - YouTube embed:

Если видео размещены на YouTube:

```jsx
<iframe
  width="100%"
  height="200"
  src="https://www.youtube.com/embed/VIDEO_ID"
  frameBorder="0"
  allowFullScreen
  style={{ borderRadius: 8 }}
/>
```
